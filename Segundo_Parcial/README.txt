/////////////////// SYSTEM OF EQUATIONS ///////////////////

TO USE THE SYSTEM OF EQUATION SOLVERS YOU NEED TO LOAD YOUR
MATRIX IN THE 'system_eq.txt' FILE

THE FILE USES THE FOLLOWING FORMAT (INCLUDING COMMAS):
	
	n
	a11,	a12,	a13,	b1
	a21,	a22,	a23,	b2
	a31,	a32,	a33,	b3	  

n = GRADE OF MATRIX (2,3,4,etc.)

FOR EXAMPLE: 
	THE FILE THAT REPRESENTS THE EQUATIONS:
	9*a0 + 8*a1 + 10*a3 = 4
	2*a0 + 5*a1 + 3*a3 = 7
	6*a0 + 4*a1 + 3*a3 = 6

	IS THE FOLLOWING:

	3
	9,8,10,4
	2,5,3,7
	6,4,3,6

ONCE YOU HAVE YOUR FILE SET UP, YOURE READY TO GO!

////////////////////// INTERPOLATION //////////////////////

TO USE THE INTERPOLATION METHODS YOU NEED TO LOAD THE DATA
INTO THE 'interpolation.txt' FILE

THE FILE USES THE FOLLOWING FORMAT (INCLUDING COMMAS):


n,r
x0,y0
x1,y1
x2,y2
.
.
.
xn,yn

n = NUMBER OF X POINTS
r = NUMBER YOU WANT TO INTERPOLATE

FOR EXAMPLE:
	THE FILE THAT RESPESENTS THE DATA:

	x= 1, y= 4
	x= 5, y= 3
	x= 6, y= 6

	r (to inteprolate) = 2

	IS THE FOLLOWING:

	3,2
	1,4
	5,3
	6,6

ONCE YOU HAVE YOUR FILE SET UP, YOURE READY TO GO!

	
////////////////////// PROGRAM //////////////////////

TO USE THE PROGRAM USE THE FOLLWING INSTRUCTIONS:

- OPEN YOUR TERMINAL AND NAVIGATE TO THE DIRECTORY IN 
WHICH THE PROGRAM IS LOCATED

- IN YOUR TERMINAL TYPE THE FOLLOWING COMMANDS 
(omit the '$')

	$gfortran -o second_partial second_partial.f90

	#THIS COMMAND WILL COMPILE THE FILE AND CREATE ANOTHER ONE YOU CAN
	EXECUTE

	$./first_partial

	#THIS COMMAND WILL EXECUTE THE PROGRAM.
